Task 1

Create todo list that store all it’s values in local storage. You should have an ability to do next actions :

1. Create todo
2. Delete todo
3. Mark todo as resolved
4. Edit todo (edit todo on doubleClick, save on enter button, cancel all changes on escape button)
5. Delete all resolved todos
6. Filter todos by “all\active\resolved”

Advanced :
1. Change position of todo. Can be implemented with two arrows that switch position of todo with todo below and
above, but it’s better to implement it with drag&drop.
